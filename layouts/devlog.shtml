Blockeditor will be:

  • A framework for building fast cross-platform collaborative applications with offline capability
  • An app to store all your documents, like Obsidian, Notes, Keep, or Notion and a server for sync

Blockeditor will contain:

  • A text editor for programming in, like VSCode
  • A WYSIWYG markdown editor for markdown files and within Zig documentation comments
  • An immediate mode GUI framework that handles designing fancy layouts similar to and in some ways better than Tailwind CSS
  • Sandboxing, so you can install any editor from any source safely
  • Desktop (Mac, Windows, Linux), Mobile (iOS, Android), and web support
